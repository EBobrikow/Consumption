# Consumption
Программа була створена для розрахунку розходу кількості натурального палива (дров) для наметів типу УСТ-56 та УСБ-56. 
Для функціонування цієї программи потрібен звичайний інтернет, адже для утримання погодних данних робиться HTTP запит до ресурсу Open-Meteo.
У зв'язку із швидким створенням застосунку не була створена система сповіщень помилок. Одже у разі винекнення проблем перевірте правильність введення вхідних параметрів
(дріб вводиться через кому 0,8), правильні дати, та підключення до інтернету.
Вихідний код застосунку доступний за посиланням https://github.com/EBobrikow/Consumption
Застосунок не несе комерційної цінності, та був створенний лише для особистого користування.
